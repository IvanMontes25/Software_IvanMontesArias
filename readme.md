# GymBodyTrainingEST

Sistema web para gestión de gimnasio desarrollado en PHP y MySQL.

## Tecnologías utilizadas

- PHP
- Html
- CSS
- MySQL
- XAMPP
- Bootstrap
- JavaScript
- n8n

## Instalación

1. Clonar el repositorio
https://github.com/IvanMontes25/GymBodyTrainingEST.git

2. Copiar la carpeta al servidor local
C:\xampp\htdocs\

3. Importar la base de datos

- Abrir **phpMyAdmin**
- Crear una base de datos llamada:
gymbodytraining

- Importar el archivo:
core/bootstrap.php
 
Abrir el sistema en el navegador

http://localhost/GymBodyTrainingEST
